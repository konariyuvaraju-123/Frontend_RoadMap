Frontend Development

Frontend development is the development of the graphical userInterface of a website, through the use of HTML, CSS, JavaScript, so that users can view and interact with that website.

  • HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML


  • Library: A Library is a set of pre-written code that you can use to add specific features to your project, saving time and effort. You don’t need to figure out how everything works internally—you just use the parts you need

  • Framework: A framework is like a ready-made toolbox that helps developers build websites or applications more easily. It provides pre-written code, rules, and structures to save time and effort. Instead of starting from scratch, you can use the tools in the framework to build things faster and in an organized way

  • Preprocessor: A preprocessor is like a helper that makes your work easier. Imagine you’re writing a long recipe, but instead of repeating ingredients or steps, you can use shortcuts or reusable notes. The helper then turns those shortcuts into the full recipe everyone can understand

_____________________________________________________________Difference Between Library and Framework_____________________________________________________________

Feature Library Framework
Definition A collection of pre-written code that you can use to perform specific tasks. A structured tool that provides rules and guidelines for building your project.
Control You are in control and call the library functions when needed. The framework is in control, and you follow its structure and rules.
Analogy A recipe book where you choose and use specific recipes as needed. A cooking kit that provides tools and instructions to guide the entire process.
Flexibility More flexible—you can pick and choose what to use. Less flexible—it provides a predefined structure.
Examples React.js, Lodash, jQuery Angular, Django, Ruby on Rails
